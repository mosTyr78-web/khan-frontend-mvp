## Description

KHAN est une application de fitness gamifiee qui permet aux utilisateurs de suivre leurs entrainements, relever des defis, et debloquer des recompenses uniquement par le merite. Aucun achat ne permet de debloquer du contenu - tout se merite.

### Programmes disponibles
- **Standard** - Workouts pour tous niveaux
- **Kids** - Exercices ludiques pour enfants (6-12 ans)
- **Mama Strong** - Programme dedie aux femmes et mamans
- **Accessible** - Exercices adaptes pour personnes a mobilite reduite
